# Cyber Threat Intelligence Reports

Automated daily threat intelligence reports generated by Claude Code's cyber-threat-intel skill.

## Reports

Reports are generated daily at 8:00 AM EST and stored in the `reports/` directory.

## Report Contents

Each report includes:
- **Critical Vulnerabilities**: CISA KEV additions, CVSS 9.0+ CVEs
- **Exploits & Zero-Days**: New exploits, actively exploited vulnerabilities
- **Malware & Ransomware**: Active campaigns, ransomware group activity
- **Threat Actors**: APT/nation-state activity, criminal groups
- **Vendor Advisories**: Microsoft, Apple, Google, SAP, VMware updates
- **Industry News**: Data breaches, security incidents
- **Recommended Actions**: Prioritized remediation guidance

## Sources

Intelligence is gathered from:
- CISA Known Exploited Vulnerabilities (KEV) Catalog
- NIST National Vulnerability Database (NVD)
- Security news outlets (BleepingComputer, The Hacker News, SecurityWeek)
- Vendor security advisories
- Threat intelligence feeds

## Automation

Reports are automatically generated and pushed via macOS launchd scheduler.
